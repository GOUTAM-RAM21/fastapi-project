# CarValue AI — React Frontend

Premium React frontend for the Car Price Prediction API.

## Setup

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Auth

- **JWT**: username `admin` / password `secret` (or your backend credentials)
- **API Key**: `demo-key`

## Build

```bash
npm run build
```

## Stack

- React 18 + Vite
- Tailwind CSS + Framer Motion
- react-hook-form + Zod
- Recharts, Lucide React, react-hot-toast
- React Router v6
